# 계산기 만들기

## 프로젝트 소개
콘솔 기반 사칙연산 계산기 프로그램

사용자로부터 두 정수와 연산자를 입력받아 계산을 수행,
연산 결과를 컬렉션에 저장하고 관리

## 사용 기술
언어: Java
개발환경: IntelliJ

## 주요기능
- 사칙연산 지원
- 0으로 나누기 예외 처리
- 잘못된 연산자 입력 처리
- 연산 결과 누적 저장
- 가장 오래된 결과 삭제 기능
- 프로그램 반복 실행 및 종료 기능

## 개선 예정
- 연산자 Enum 적용
- 제너릭 활용한 double 연산 지원
- Lambda&Stream 이용한 연산 결과 조회 기능
